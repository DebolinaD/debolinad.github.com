If I speak about Stack, the first thing that comes to my mind is LIFO: Last in First Out.

 A stack is actually an abstract data type and a linear data structure and is characterized by two basic operations push and pop.

Push symbolizes adding a new element to the existing set or initialing a new stack if it is empty and pop means retrieving the last element from the stack or nothing if the stack is empty.

Someway it is more like a stack of plates in a canteen, where each plate is stored above the other and if one tries to remove it then one needs to pullout the one at the top.
