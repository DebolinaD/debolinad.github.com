Faster Copy-Paste/Cut-Paste for Files

Published: 22 Feb 2012

 One command, that am sure everyone has used more than any other thing is Copy-Paste/Cut-Paste.

Imagine how your life would be, if you couldn’t copy a file from

one location to other?

That means you can’t borrow movies from friends, not even games  and of course no code copying from Google.

 Tough Life I must say.

    However even though its really important, the process that

almost everyone follows is 

1.  Open a windows explorer. 2.  Traverse to the file location 3.  Right click on it or select copy or may be Ctrl+C (to copy)  or select cut or use Ctrl+X (to cut)   4. then again go to the destination(where you 

want to paste) 5. right click to select the option paste or use Ctrl+V

The easiest solution for this is cp for Copy-Paste and mv for Cut-Paste. 

This sorts the whole thing in a single line.
